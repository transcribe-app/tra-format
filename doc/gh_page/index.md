  <h2>About the Project</h2>
  <p>
    The <strong>TRA Format</strong> repository contains the open specification for the files generated by the <a href="https://transcribe.wreally.com/">Transcribe App</a>. This format allows for the structured storage of speech-to-text data, including:
  </p>
  <ul>
    <li>Precise timestamps for word-level synchronization.</li>
    <li>Speaker identification and labeling.</li>
    <li>Edit history and metadata.</li>
  </ul>

  <h3>For Developers</h3>
  <p>
    This repository provides a reference implementation written in <strong>Flutter</strong>. It includes example of a viewer/player widget, making it easier to integrate <code>.tra</code> file support into your own applications or to convert data for custom workflows.
  </p>
